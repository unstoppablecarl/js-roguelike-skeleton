loadEntitiesFromArrayString

(
  • mapData
  • charToType
  • [replaceCurrentEntities=false]
)

Defined in src/entity-manager.js:134

Loads Entity data from an array of strings.

Parameters:

  • mapData Array
    • The array of strings to load.
  • charToType Object
    • An object mapping string characters to Entity types (see Entity.Types[type]). Characters in mapData not in charToType are ignored.
  • [replaceCurrentEntities=false] Bool optional
    • If true current entities at positons of entities being added will be removed. Otherwise new entities at occupied positions will be skipped.

Example:

       // 'P' will be ignored and a floor tile will be placed at that position
       var mapData = [
           '####',
           '#..#',
           '#.Z#',
           '####',
       ],
       charToType = {
           'Z': 'zombie'
       };

       entityManager.loadTilesFromArrayString(mapData, charToType);
